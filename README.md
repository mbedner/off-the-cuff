# Off the Cuff

A mobile-first speaking-practice web app for short improv-inspired weekday drills.

## Run locally

This first version uses Tailwind and Remix Icon through browser CDNs and does not need a build step.

Open `index.html` in a browser, or serve the folder with any static web server. Audio recording requires a secure browser context, so it works best from `localhost`.

## Included

- Today workflow with a lesson, 2 quick drills, timer, rating, reflection, and completion
- Practice tab with 12 modes
- Progress stored in `localStorage`
- Lesson bank with 25 seeded lessons
- Settings for session length and prompt types
- Panic button with quick speaking frameworks
- Optional audio recording and playback when the browser supports it
